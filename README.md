# TON Conservative Meme Bot

Bot di trading conservativo per meme coin su TON blockchain.

## Deploy su Render

1. Configura variabili ambiente:
   - TELEGRAM_BOT_TOKEN
   - TELEGRAM_CHAT_ID  
   - MNEMONIC_WORDS
   - PORT

2. Deploy automatico da GitHub

## Comandi Telegram

- /status - Status bot
- /stats - Statistiche
- /positions - Posizioni aperte
- /help - Aiuto
